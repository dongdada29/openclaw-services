### 2. 无限循环无退出机制 (Watchdog)

**文件**: `services/watchdog/index.js`
**位置**: Line 230

```javascript
while (true) {
  // ... 监控逻辑
  await new Promise(r => setTimeout(r, currentInterval));
}
```

**问题**: `while(true)` 循环没有退出机制，无法优雅关闭。

**修复建议**:
```javascript
let running = true;

process.on('SIGINT', () => { running = false; });
process.on('SIGTERM', () => { running = false; });

while (running) {
  // ... 监控逻辑
  await new Promise(r => setTimeout(r, currentInterval));
}
```

### 3. pkill 可能误杀其他进程

**文件**: `cli/src/index.js` Line 247, `services/watchdog/index.js` Line 137

```javascript
execSync('pkill -f "(node|bun).*model-proxy"', { stdio: 'pipe' });
```

**问题**: 宽松的匹配模式可能误杀其他包含 `model-proxy` 字符串的进程。

**修复建议**:
```javascript
execSync('pkill -f "node.*openclaw.*model-proxy/server.js"', { stdio: 'pipe' });
// 或使用精确 PID 杀掉
```

### 5. XML 转义不完整

**文件**: `cli/src/index.js`
**位置**: Line 456

```javascript
const escapeXml = (str) => str.replace(/&/g, '&amp;').replace(/</g, '&lt;').replace(/>/g, '&gt;').replace(/"/g, '&quot;');
```

**问题**: 没有处理 `'` 单引号，在某些 XML 上下文中可能有问题。

**修复建议**:
```javascript

  .replace(/&/g, '&amp;')
  .replace(/</g, '&lt;')
  .replace(/>/g, '&gt;')
  .replace(/"/g, '&quot;')
  .replace(/'/g, '&apos;');
```
